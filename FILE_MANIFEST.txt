  • Defines build output directory
  • Sets dev server port
  • Specifies React plugin usage

tailwind.config.js
  • Tailwind CSS customization
  • Defines color schemes
  • Extends default themes
  • Configures plugins

postcss.config.js
  • PostCSS processing pipeline
  • Enables Tailwind + Autoprefixer
  • Optimizes CSS output

index.html
  • HTML template for React app
  • Defines page metadata & SEO tags
  • Points to src/main.jsx

  • Specifies build command

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

"npm: command not found"
  → Install Node.js from nodejs.org

"Port 3000 already in use"
  → Change port: npm run dev -- --port 3001

"Module not found"
  → Clear cache: rm -rf node_modules && npm install

"Build fails"
  → Check build logs in Vercel/Netlify dashboard
  → Try: npm run build locally first

"404 on page refresh"
  → Fixed in vercel.json and netlify.toml (already configured)
