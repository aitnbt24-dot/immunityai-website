✅ STEP-BY-STEP TO LIVE WEBSITE (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

1️⃣  SETUP (30 seconds)
   └─ Open PowerShell in this folder
   └─ Run: npm install

2️⃣  TEST LOCALLY (30 seconds)
   └─ Run: npm run dev
   └─ Visit: http://localhost:3000
   └─ Press Ctrl+C to stop

3️⃣  CREATE GITHUB REPO (1 minute)
   └─ Go to github.com/new
   └─ Name: immunityai-website
   └─ Copy HTTPS URL

4️⃣  PUSH CODE (1 minute)
   └─ Run: git init
   └─ Run: git add .
   └─ Run: git commit -m "Initial commit"
   └─ Run: git remote add origin <PASTE-URL>
   └─ Run: git branch -M main
   └─ Run: git push -u origin main

5️⃣  DEPLOY TO VERCEL (1 minute)
   └─ Go to vercel.com/new
   └─ Import immunityai-website
   └─ Click Deploy
   └─ Wait 1-2 minutes

✅ SITE IS LIVE!
   └─ Visit: https://immunityai-website.vercel.app

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

"Node.js not found"
→ Install from nodejs.org

"npm: command not found"
→ Restart PowerShell after installing Node.js

"package.json not found"
→ Make sure you're in the correct folder
→ Should see package.json when you run: ls

"Build fails"
→ Run: npm install (to reinstall dependencies)
→ Run: npm run build (to test locally)

"Port 3000 already in use"
→ Run: npm run dev -- --port 3001

"GitHub repo not found"
→ Make sure repository is PUBLIC (not private)
→ Verify HTTPS URL is correct

See GITHUB_DEPLOYMENT_START_HERE.md for more solutions!
